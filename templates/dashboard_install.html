安装主管看板

把下面 3 个文件复制到 Google Apps Script → 部署成 Web 应用 → 拿到看板 URL

▸ 安装步骤（5 分钟）

  1. 打开你的 Google Sheet {% if sheet_id %} → 打开 {{ company_display or "当前 Sheet" }} {% endif %}
  2. 菜单 扩展功能 → Apps Script，会开一个新分页
  3. 依序复制以下 3 个文件到 Apps Script 编辑器：
    • 左侧预设的 Code.gs → 全选删掉 → 贴下面「① Code.gs」的内容
    • 点左侧 + → HTML → 命名 Dashboard（**不要加 .html**）→ 贴「② Dashboard.html」内容
    • 左下 ⚙ 设定 → 勾「在编辑器中显示 appsscript.json 资讯清单档」→ 回编辑器会多出 appsscript.json → 贴「③ appsscript.json」内容
  4. 顶部 部署 → 新增部署项目 → 类型选 网路应用程式
    • 执行身分：我本人
    • 存取权：只限我本人（或 知道连结的任何人 — 看你要不要给外人看）
    • 点「部署」→ 授权 → 拿到一个 /exec 结尾的 URL
  5. 把 URL 加到浏览器书签就是你的看板入口
⚠ 已装过想更新代码：Apps Script 编辑器里重贴 Code.gs → 右上部署 → 管理部署项目 → 右上铅笔 → 版本选「新版本」→ 部署。URL 不会变。
